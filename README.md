Descripción
Este proyecto es una API, desarrollada con Node.js, Express, y MongoDB. La API permite realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) sobre los usuarios, productos, y agregar productos al carrito además de manejar la autenticación mediante JWT (JSON Web Token) y la autorización de usuarios basados en roles (administradores, compradores y vendedores).

La arquitectura del proyecto sigue un patrón Three-Tier Architecture (Arquitectura de Tres Capas), estructurada en las capas de Presentación, Lógica de Negocio y Acceso a Datos, asegurando una separación clara de responsabilidades y facilitando la escalabilidad.
